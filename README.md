SAFINA CARPET – Full Stack AI Powered E-Commerce Platform
Overview
SAFINA CARPET is a full-stack MERN based e-commerce platform built for showcasing and selling premium carpets online.
The project includes:
Customer-facing e-commerce website
Separate Admin Dashboard
AI-powered backend modules
Product and order management system
Authentication & role-based access
Visualization and chatbot integrations
The platform is designed with scalability, modern UI/UX, and future AI integrations in mind.
